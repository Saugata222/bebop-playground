What it does

A fully working Copilot shell prototype with 17 components — all generated from design tokens and interactive out of the box.

  • Interactive shell nav, header, chat input, add menu, sources, send → thinking → streaming response
  • Component catalog 10 compound + 6 primitive components, each with live preview and state showcase
  • Explorations fork the shell into named explorations and iterate independently
  • AI-ready prompt Copilot or Claude to modify the prototype. Project context is provided automatically
  • Zero dependencies every preview is a single HTML file. Download, double-click, done
