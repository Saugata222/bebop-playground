Get Started with Bebop Shell

Set up the project locally and use an AI coding assistant to build new features on top of the Copilot Shell prototype.

1
Install prerequisites Download and install: • Node.js (v18 or later) • VS Code • Claude Code or GitHub Copilot extension for VS Code
2
Get the project Download the zip or clone from GitHub:
Download the zip and extract it into a folder: Download from OneDrive
git clone https://github.com/sapramanik_microsoft/bebop-playground.git
3
Install dependencies Open the project folder in VS Code, then install:
npm install
"Install the project dependencies"
"Run npm install to set up this project"
4
Start building Prompt the AI to modify the shell. It reads CLAUDE.md automatically for project context. Example prompts:
"Add a citations panel below the response text in the shell"
"Add file attachment chips above the chat input"
"Create a settings dialog with toggle switches for shell preferences"
5
Build and preview After the AI makes changes, rebuild the shell and view the result:
npm run build
Then open preview/dist/shell.html in your browser.
"Build the shell preview and tell me where the output file is"
Claude will run npx tsx preview/src/shell.ts and tell you the output path.
"Run the build script and open the shell preview"
Quick reference
npm run build — rebuild shell only
npm run build:all — rebuild all previews
npm run serve — start server on port 8765
npm run dev — build + start server
npx tsc --noEmit — type-check
